大部分位置的價值變成 -10，只有終點是 0，背後的意義與數學原理如下：

1. 為什麼會是 -10？（無限迴圈的結果）
這跟強化學習 (Reinforcement Learning) 的策略評估 (Policy Evaluation) 以及我們目前使用的「隨機指定單一方向」的策略有關。
* 您可以看一下旁邊的 Policy Matrix，每個格子都被隨機指派了一個固定的箭頭方向。
* 這種「無腦隨機亂指」的走法，絕大部分的格子最後都會引導 Agent 去撞牆、撞障礙物（留在原地），或是讓兩個格子互相指著對方（例如 A 往右走、右邊的 B 往左走，形成無限迴圈）。
* 在強化學習中，我們通常設定每走一步的懲罰（Reward）是 -1。當 Agent 陷入無限死胡同，它就會在每一回合不斷收到 -1。
* 為了讓數值收斂，程式有加上折扣因子（Discount Factor γ = 0.9）。在無窮迴圈下，這個狀態的價值可以用數學的無窮等比級數計算：
  V(s) = -1 + 0.9(-1) + 0.9^2(-1) + ... = (-1) / (1 - 0.9) = -10
* 因此，只要是順著箭頭走永遠到不了終點的格子，其價值最後一定會算出來剛剛好等於 -10.0。

2. 為什麼終點是 0？
因為終點 (Goal) 是一個目標狀態 (Terminal State)，Agent 一旦抵達這裡回合就結束了，不會再有任何後續的步驟或懲罰被扣分，所以它本身的未來預期價值就是 0。如果有任何一個格子運氣好，它的一連串箭頭剛好可以直接指向終點，它的價值就不會是 -10（例如走一步就到終點的格子會是 -1），但在這麼大的地圖隨機產生的狀況下，能剛好指到終點的機率非常低，所以您才會看到滿滿的 -10。

是的！您的理解完全正確！ 💯

這在強化學習 (Reinforcement Learning) 的領域中，正式名稱叫做 確定性策略 (Deterministic Policy)。

您可以把它想像成：在一張大富翁或是迷宮的地圖上，我們預先在每一個格子上用麥克筆畫上了一個「強制單行道」的箭頭。
未來不管 Agent（走迷宮的人）是因為什麼緣故、從哪邊走到這個格子，它到了那裡就不需要再思考了，也沒有其他選擇，只能「100% 乖乖聽話，照著這個格子畫好的箭頭方向繼續走」。

這也就是為什麼價值會算成 -10 的根本原因：
由於這些箭頭是「一開始就亂指並固定下來的」，所以您可以看著您剛剛截圖的 Policy Matrix 想像一下：如果 Agent 把一隻腳踩在任何一個隨機箭頭上開始走，它幾乎一定會碰到以下兩種狀況：
1. 撞牆 (死胡同)：走到一個箭頭朝向牆面、或是朝向灰色障礙物的格子。它照著箭頭走，只能不斷「撞牆、留在原地、被扣 1 分」，永遠無法離開。
2. 無限迴圈 (踢皮球)：走到兩個格子箭頭剛好互指（或是繞一圈回到原點的格子）。Agent 就會在這個迴圈裡無限繞圈圈，也是永遠走不到終點。

因為 Agent 在這兩種狀況下都永遠到不了 0 的終點，只能一直被扣分，經過折現運算收斂後，那個格子的未來預期價值就變成了精準的 -10 喔！您的觀察銳利而且非常精確地抓到了策略評估的核心意義！
